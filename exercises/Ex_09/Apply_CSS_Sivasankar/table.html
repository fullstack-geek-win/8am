Table & List

table : represent data in the form of table

thead(table header) : the rows that consist of the column headers for the table.

tbody(table body) : the rows that consist of body of data for the table.

tr(table row) : a row of cells in a table.

td(table cell) : represents a table cell in a table.

rowspan : sets how many rows a cell will span.
colspan : sets how many columns a cell will span.

Tag Type Attributes Value/Description
table table
thead table head
tbody table subject
tr table row
td table data
rowspan
colspan

ul(unordered list) : a list of items, where the order of items is not important

li(list items) : represent a list items of an ordered or unordered list.

ol(ordered list) : a list of items, where the items are intentionally ordered.

dl(data list) : tag specifies a list of pre-defined options for an "input" element.

Tag Type Attributes Value/Description
ul unordered list
type square, box
li list items
ol ordered list
type A, a, l, i
start start from number
dl data list
dt data term
dd data definition
